本自主智驾小车采用后轮驱动 + 前轮阿克曼转向四轮底盘架构；主控采用 ESP32-S3，搭配 OpenMV N6视觉主板完成红绿交通立柱、泊位标线识别，辅以多传感器融合感知赛道边界、车身姿态、障碍物距离；整机硬件分层模块化布局，软件采用有限状态机实时定位、视觉识别、转向控制、泊车逻辑，所有外设通过 UART/I2C/GPIO 标准总线与主控通信。
